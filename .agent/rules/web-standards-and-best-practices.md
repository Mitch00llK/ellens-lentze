Webstandaarden en Best Practices
HTML & Semantiek

Semantic HTML5 met proper heading hierarchy (h1-h6)
Enkel één h1 per pagina
Logische heading nesting zonder sprongen
Correct gebruik van <main>, <nav>, <article>, <section>
Native HTML elements boven divs waar mogelijk

Accessibility (WCAG 2.2 Level AA)

ARIA labels voor interactieve elementen
Keyboard navigation volledig ondersteund
Color contrast minimaal 4.5:1 voor tekst
Focus indicators zichtbaar (niet verwijderd)
Alt text op alle afbeeldingen
Form labels correct gekoppeld
Skip links waar relevant

CSS Moderne Technieken

Flexbox voor 1D layouts
CSS Grid voor 2D layouts
CSS Custom Properties (variabelen) verplicht
Mobile-first responsive design
Fluid typography met clamp()
CSS transforms voor animaties (GPU-acceleratie)

Core Web Vitals Optimalisatie

LCP (Largest Contentful Paint) < 2.5s
FID (First Input Delay) < 100ms
CLS (Cumulative Layout Shift) < 0.1

SEO & Schema Markup

JSON-LD schema voor structurele data
Microdata waar van toepassing
Open Graph meta tags
Breadcrumb schema in navigatie
